Support

This page provides how to use this application and application support information.

How to use

■ Home Screen

① By touching this button, the Calendar Screen will be displayed.

② When you want to search diary, you just need to touch this button to move to the Search Screen.

③ The diary will be displayed in a timeline from the present to the past from top to bottom when this button is touched.

④ The diary will be displayed randomly with any time from top to bottom when this button is touched.

⑤ When this button is touched, the screen to create a new diary will be displayed.

■ Calendar Screen

① When touching the arrows, the month and year will be increased or decreased by 1 unit.

② If you touch the Today button, the Calendar will go back to the current date and show all the diary of that day.

③ The days with the blue dot below are the days with the diary, the days without the blue dot mean that the day does not have any diary. You can choose any date you want, then all the diary of the date you choose will be displayed at the bottom.

④ The section that displays the entire diary of the selected date.

⑤ You can also move to the new diary creation screen by touching this button.

■ Search Screen

① Textbox input search condition. When inputting this textbox, any diary that contains the input value will be displayed below.

② Textbox input search condition. When inputting this textbox, any diary that has a tag containing the input value will be displayed below.

③ Select color to search. When choosing a color, any diary that has the same background as the selected color will be displayed below.

④ Favorite search. When turn to right (on), all diary marked as farovite will be displayed lower.

⑤ The display of search results.

■ Create Diary Screen

① This textbox is using for input diary content.

② This textbox is using for input tag content.

③ This color range is used to select the background color for the textbox content above. When choosing a color, the background will change to that color. After saving the diary, the selected color will also be displayed on pages such as home, search and calendar screen.

④ You can select an image from your Camera Roll to attach it to your diary content by touching this button.

⑤ If you want to take a photo or record a video, just touch this button and then choose to record video or take a picture. Photos or videos after being recorded will be attached to diary content.

⑥ You can show or hide the range of colors by touching this button.

⑦ You can show or hide the input textbox tag range by touching this button.

⑧ After you finished input content, tag, image for diary you can touch this button to save.

■ Setting Screen

① You can ask for a password when opening this application by turning right (on) this and input your password.

② When you want to set what day of the week is the first column of the calendar used in this application, you just have to choose here.
